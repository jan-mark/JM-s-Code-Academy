Introduction to C

C is a powerful and fast programming language created by Dennis Ritchie in 1972. It is widely used for building operating systems, software, and embedded systems. A low-level, general-purpose programming language that provides direct access to memory and hardware, making it efficient for system programming, embedded systems, and performance-critical applications. C is the foundation of many modern programming languages and is commonly used in operating systems and game development.

Basic Syntax of C

- Every C program starts with `#include `.
- The `main()` function is the entry point.
- Statements end with `;`.
- Curly braces `{}` define code blocks. 

                
                #include 
                int main() {
                    printf("Hello, World!");
                    return 0;
                }

Variables and Data Types

Variables store values in memory. Common data types:
- `int` (whole numbers)
- `float` (decimal numbers)
- `char` (single character)
- `double` (large decimal numbers) 

                
                int age = 20;
                float price = 10.5;
                char grade = 'A';

Operators in C

Operators perform operations on values.
- Arithmetic (`+`, `-`, `*`, `/`, `%`)
- Relational (`==`, `!=`, `>`, `<`, `>=`, `<=`)
- Logical (`&&`, `||`, `!`) 

                
                int a = 10, b = 5;
                int sum = a + b;   // Addition
                int isEqual = (a == b);  // Relational Operator

Loops in C

Loops execute code multiple times:
1. For Loop - Runs a fixed number of times.
2. While Loop - Runs while a condition is true.
3. Do-While Loop - Runs at least once. 

                
                for(int i = 1; i <= 5; i++) {
                    printf("%d ", i);
                }
