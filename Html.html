Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages.HTML defines the elements of a webpage, such as text, images, links, and multimedia, forming the foundation of web development. It works alongside CSS and JavaScript to enhance the design and functionality of websites.

HTML Elements

HTML elements define the structure of a webpage.

Example:


<h1>Hello, World!</h1>
<p>This is a paragraph.</p>

HTML Attributes

Attributes provide additional information about elements.

Example:


<a href="https://www.google.com" target="_blank">Visit Google</a>

HTML Forms

Forms collect user input.

Example:


<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

HTML Tables

Tables organize data in rows and columns.

Example:


<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>
