Introduction to JavaScript

JavaScript (JS) is a powerful programming language used to make websites interactive. It enables features like animations, real-time updates, interactive forms, and client-side validation. JavaScript works alongside HTML and CSS and can also be used for backend development with frameworks like Node.js.

Basic Syntax of JavaScript

JavaScript is case-sensitive and uses semicolons (`;`) to end statements. Code blocks are enclosed in `{}`.


console.log("Hello, World!");  // Prints text to the console

Variables and Data Types

JavaScript variables can be declared using `var`, `let`, or `const`.


let age = 25;       // Number
double price = 99.99; // Float
let name = "JM";    // String

Operators in JavaScript

JavaScript supports various operators:

  • Arithmetic: `+`, `-`, `*`, `/`, `%`
  • Comparison: `==`, `!=`, `>`, `<`, `>=`, `<=`
  • Logical: `&&`, `||`, `!`

let x = 5 + 3;  // Addition
let isEqual = (x === 8);  // Strict Comparison

Loops in JavaScript

JavaScript provides `for`, `while`, and `do-while` loops.


for (let i = 1; i <= 5; i++) {
    console.log(i);  // Prints numbers 1 to 5
}
