Introduction to PHP

PHP (Hypertext Preprocessor) is a server-side scripting language used to build dynamic websites and web applications. It allows developers to create dynamic and interactive websites, process form data, manage sessions, and interact with databases like MySQL. PHP is often used in content management systems (CMS) like WordPress.

Basic Syntax of PHP

PHP code is embedded within HTML and starts with `<?php` and ends with `?>`.


<?php
    echo "Hello, World!";
?>

Variables and Data Types

PHP variables start with a `$` symbol and do not require explicit data types.


$age = 25;         // Integer
$price = 99.99;    // Float
$name = "JM";      // String
$is_logged_in = true;  // Boolean

Operators in PHP

PHP supports various operators:

  • Arithmetic: `+`, `-`, `*`, `/`, `%`
  • Comparison: `==`, `!=`, `>`, `<`, `>=`, `<=`
  • Logical: `&&`, `||`, `!`

$x = 5 + 3;  // Addition
$is_equal = ($x == 8);  // Comparison

Loops in PHP

PHP provides `for`, `while`, and `foreach` loops.


for ($i = 1; $i <= 5; $i++) {
    echo $i . " ";  // Prints numbers 1 to 5
}
