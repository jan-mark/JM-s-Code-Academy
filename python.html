Introduction to Python

Python is a simple, powerful, and beginner-friendly programming language created by Guido van Rossum in 1991. A high-level, interpreted programming language known for its simple syntax and readability. It is widely used in web development, data science, artificial intelligence, automation, and software development. Python’s extensive libraries and frameworks make it a popular choice for beginners and professionals alike.

Basic Syntax of Python

Python uses indentation instead of curly braces `{}` to define code blocks.


                print("Hello, World!")  # Prints text to the screen

Variables and Data Types

Variables in Python do not require explicit data types. The type is inferred.


                age = 20         # Integer
                price = 10.5     # Float
                name = "JM"      # String

Operators in Python

Python supports arithmetic, relational, and logical operators.


                x = 5 + 3  # Addition
                is_equal = (x == 8)  # Comparison

Loops in Python

Python has `for` and `while` loops for iteration.


                for i in range(1, 6):
                print(i)  # Prints numbers 1 to 5
